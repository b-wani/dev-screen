전역 단축키 ⌥⌘R카피 갱신

⌥⌘R 을 누르면 어디서든 캡처 툴바를 불러와요. 녹화 중에 누르면 바로 정지합니다. 어떤 앱을 쓰고 있든 동작해요.

Recap은 메뉴바에 상주해요. 창을 닫아도 종료되지 않고, 메뉴바 아이콘 → Welcome 다시 보기로 이 안내를 언제든 다시 열 수 있어요.

이 목업이 보여주는 것

셸만 재설계. 좌 레일(마스코트 히어로 + 챕터 내비)과 하단 바가 신규 셸. 우측 본문은 기존 OnboardingStepBody 콘텐츠를 그대로 마운트한 모습.

챕터 내비 = 사이드바(권장안). 7단계가 항상 보이고 현재 단계 강조, 완료는 ✓. 라벨은 ONBOARDING_STEPS[].title 그대로.

카피 갱신: ⌥⌘R = "녹화 시작/정지" → "캡처 툴바 소환(녹화 중이면 정지)"로 수정. "Welcome 다시 보기" 재소환 경로 안내 추가.

모든 색·radius·폰트는 Hoppy C-dark 토큰. 키/번호는 SF Mono, 헤딩은 SF Pro Rounded. 마스코트는 히어로에만 — 권한 단계 본문 등 기능 크롬에는 넣지 않음.
