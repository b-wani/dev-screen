# 창별 화면 상세 스펙 (Hoppy 재설계, 맵 #52)

[웨이파인더 맵 #52](https://github.com/b-wani/recap/issues/52)의 창 스펙 티켓 산출 자산이다. 각 스펙은 확정된 새 디자인 언어([../hoppy-design-language.md](../hoppy-design-language.md), C-dark)를 적용한 초안 + C-dark 목업이다.

| 파일 | 티켓 | 화면 |
|---|---|---|
| `57-capture-toolbar-overlay.md` / `.mockup.html` | [#57](https://github.com/b-wani/recap/issues/57) | 플로팅 캡처 툴바 · 선택 오버레이 |
| `58-editor.md` / `.mockup.html` | [#58](https://github.com/b-wani/recap/issues/58) | 에디터(프리뷰 + 우측 패널 + 타임라인) |
| `60-library.md` / `.mockup.html` | [#60](https://github.com/b-wani/recap/issues/60) | 라이브러리 창(썸네일 그리드) |
| `61-welcome-shell.md` / `.mockup.html` | [#61](https://github.com/b-wani/recap/issues/61) | Welcome(온보딩) 셸 |

각 초안의 "열린 결정(open decisions)"은 해당 이슈의 **해소 코멘트에서 확정**됐다 — 확정 값은 이슈 코멘트가 authoritative. 구현(코드 전면 수정)은 맵 목적지 밖이며, 다중 창 전환 경로와 최종 구현 티켓 분해는 후속 티켓에서 다룬다.
